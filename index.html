// motor de segurança

Blindado
por design.

Não é segurança de fachada. Cada camada foi construída para resistir a ambientes onde falhas têm consequências reais.

01
Anti Brute-Force Automático

5 tentativas falhas → IP bloqueado por 5 minutos. Blacklist permanente configurável.

02
Token com IP Binding

Sessão amarrada ao IP de origem. Token usado de outro IP é detectado e bloqueado instantaneamente.

03
Criptografia BEJ-EDG-AES256

Dados em repouso protegidos com cadeia de criptografia própria. Arquivos JSON convertidos para formato .bej criptografado.

04
Isolamento por Empresa e Nível

Cada empresa tem dados completamente isolados. Agente, Supervisor, Gestor e Dono acessam apenas seu escopo.

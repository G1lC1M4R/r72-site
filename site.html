// motor de segurança

Blindado
por design.

Não é segurança de fachada. Cada camada foi construída para resistir a ambientes onde falhas têm consequências reais.

01
Anti Brute-Force Automático

5 tentativas falhas → IP bloqueado automaticamente por 5 minutos. Blacklist permanente configurável pelo administrador.

02
Token com IP Binding

Cada sessão é amarrada ao IP de origem. Token roubado e usado de outro IP é detectado e bloqueado instantaneamente.

03
Auditoria de Acessos Completa

Todo evento de segurança registrado: login, falha, bloqueio, logout — com timestamp preciso e IP real mesmo atrás de proxy.

04
Isolamento por Nível de Acesso

Agente, Supervisor e Gestor acessam apenas seu módulo. Portas isoladas, tokens distintos, zero cross-access.
